# Slider 滑块

### 基础用法

<br />
<template>
  <os-slider />
</template>
<br />
<br />
由type属性来选择tag的类型。

```vue
<template>
  <os-tag>土豆泥</os-tag>
  <os-tag type="success">雪芸豆</os-tag>
  <os-tag type="text">大闸蟹</os-tag>
  <os-tag type="warning">火腿肠</os-tag>
  <os-tag type="error">碎冰冰</os-tag>
</template>
```
